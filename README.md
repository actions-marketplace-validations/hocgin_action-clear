# action-clear

> 用于清除 tags / release / workflow

### 参数

| 参数                  | 用途                      |
|---------------------|-------------------------|
| limit_tags          | 保留最后 tags 数量，-1 全保留   |
| limit_release       | 保留最后 release 数量，-1 全保留   |
| limit_failure_workflow_run | 保留最后 workflow 错误日志数量，-1 全保留   |
